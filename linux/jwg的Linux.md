##### pip3

> **安装**
>
> >  wget https://bootstrap.pypa.io/pip/get-pip.py
> >
> > python3 get-pip.py
> >
> > pip -V
>
> **解决pip3下载很慢使用国内源**
>
> > pip3 install -i https://pypi.tuna.tsinghua.edu.cn/simple 要安装的库的名称

> python3 -m http.server 7890	
>
